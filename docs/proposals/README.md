# Proposals

This directory is the canonical intake lane for product ideas that are not yet active
product builds.

Each proposal should:

- use `docs/SKILL_PROPOSAL_TEMPLATE.md` as the base shape
- stay narrow and falsifiable
- include a build-now / keep-warm / hold / kill decision
- link back to any relevant suite or portfolio strategy note

If an idea is a portfolio pattern rather than one skill, add one strategy note plus
one proposal file per likely product lane.
